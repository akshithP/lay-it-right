# LayItRight - Tile Planning Calculator\n\nA comprehensive web application for DIY enthusiasts to plan tiling projects with precision. Calculate tile quantities, visualize layouts, and get accurate measurements for your next home improvement project.\n\n## 🚀 Features\n\n- **Precise Calculations**: Calculate exact tile quantities, area coverage, and waste percentages\n- **Multiple Units**: Support for metric (mm, cm, m) and imperial (inches, feet) measurements\n- **Layout Patterns**: Grid, brick/running bond, and herringbone patterns\n- **Visual Preview**: Interactive SVG-based tile layout visualization (coming soon)\n- **Project Management**: Save and manage multiple tiling projects\n- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices\n- **Accessibility**: WCAG 2.1 AA compliant with full keyboard and screen reader support\n\n## 🛠️ Tech Stack\n\n- **Framework**: Next.js 14 (App Router)\n- **Language**: TypeScript\n- **Styling**: Tailwind CSS\n- **UI Components**: shadcn/ui (Radix primitives)\n- **State Management**: Zustand + Immer\n- **Forms**: react-hook-form + Zod validation\n- **Animations**: Framer Motion\n- **Testing**: Jest + Testing Library\n- **Unit Conversion**: convert-units library\n\n## 📋 Project Structure\n\n```\nsrc/\n├── app/                    # Next.js app router pages\n│   ├── project/\n│   │   ├── wizard/         # Project creation wizard\n│   │   └── [id]/          # Individual project pages\n│   └── globals.css        # Global styles\n├── components/\n│   ├── ui/                # Reusable UI components (shadcn/ui)\n│   ├── wizard/            # Wizard-specific components\n│   ├── forms/             # Form components\n│   └── preview/           # Tile preview components\n├── lib/\n│   ├── utils.ts           # Utility functions\n│   └── validations.ts     # Zod schemas\n├── store/\n│   └── project-store.ts   # Zustand state management\n├── types/\n│   └── index.ts           # TypeScript type definitions\n├── utils/\n│   ├── tile-calculations.ts # Core calculation logic\n│   └── unit-conversions.ts  # Unit conversion utilities\n└── hooks/                 # Custom React hooks\n```\n\n## 🚀 Getting Started\n\n### Prerequisites\n\n- Node.js 18+ \n- npm or yarn\n\n### Installation\n\n1. Clone the repository:\n```bash\ngit clone <repository-url>\ncd lay-it-right\n```\n\n2. Install dependencies:\n```bash\nnpm install\n# or\nyarn install\n```\n\n3. Run the development server:\n```bash\nnpm run dev\n# or\nyarn dev\n```\n\n4. Open [http://localhost:3000](http://localhost:3000) in your browser.\n\n### Available Scripts\n\n- `npm run dev` - Start development server\n- `npm run build` - Build for production\n- `npm run start` - Start production server\n- `npm run lint` - Run ESLint\n- `npm run lint:fix` - Fix ESLint issues\n- `npm run type-check` - Run TypeScript type checking\n- `npm run test` - Run tests\n- `npm run test:watch` - Run tests in watch mode\n- `npm run test:coverage` - Run tests with coverage\n\n## 🧮 Calculation Features\n\n### Supported Layout Patterns\n\n1. **Grid (Straight Lay)**: Simple rectangular grid pattern\n   - Minimal waste (10%)\n   - Easy installation\n   - Best for beginners\n\n2. **Brick (Running Bond)**: Offset pattern like traditional brickwork\n   - Moderate waste (15%)\n   - Hides imperfections\n   - Classic appeal\n\n3. **Herringbone**: Zigzag pattern with tiles at 90° angles\n   - Higher waste (20%)\n   - Complex installation\n   - Premium visual impact\n\n### Calculation Logic\n\n- **Area Calculation**: Precise room area in chosen units\n- **Tile Count**: Exact number of tiles needed including waste\n- **Cut Tile Analysis**: Distinction between full and cut tiles\n- **Grout Consideration**: Accounts for grout line width in calculations\n- **Waste Management**: Pattern-specific waste recommendations\n\n## 🎨 Design System\n\n### Color
